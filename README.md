# Appfolio Listings Custom

WordPress plugin to display **Appfolio** property listings via shortcodes, with templates, slider, carousel, and map support.

- **Requires:** WordPress 5.8+, PHP 7.4+
- **License:** GPL-2.0-or-later

## Install

1. Download **`appfolio-listings-custom-*.zip`** from [Releases](https://github.com/spkldbrd/appfolio-listings-custom/releases).
2. **Plugins → Add New → Upload** and activate.
3. In wp-admin, open **Appfolio** and set your Appfolio URL, optional Google Maps API key, and listing pages.

## Shortcodes

Primary shortcode: **`[apfl_listings]`** — see plugin admin for slider, carousel, and builder options.

## Updates

The plugin reads a public **`version.json`** (default points at this repo’s `main` branch). You can publish new versions by updating that file and attaching a release ZIP to a GitHub release.
